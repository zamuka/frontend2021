Request - просьба
GET /api/json/get/ceyrBcxPOq HTTP/1.1

host - кого мы просим (www.json-generator.com)
method - действие, которое мы просим совершить GET/POST/DELETE/PUT/UPDATE
version protocol - версия протокола

headers - детали, необходимые для выполнения просьбы или уточняющие ответ.
path - ресурс, с которым мы производим действие. Строка path немного сложнее, чем мы пока посмотрели. Она также может содержать детали запроса.
body - получаемые или отправляемые данные. Бывает называют payload - полезная нагрузка.


port - часть TCP, а не HTTP - уточнение к адресу сервера.
